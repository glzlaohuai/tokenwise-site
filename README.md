# TokenWise — Website

Landing, Privacy Policy, and Support pages for **TokenWise**, a local token-usage
tracker for AI coding tools (Claude Code / Codex / OpenCode / Gemini) that lives in
the macOS menu bar.

- **Live:** https://glzlaohuai.github.io/tokenwise-site/
- Privacy Policy — [`privacy.html`](privacy.html)
- Support / FAQ — [`support.html`](support.html)

Plain static HTML with a client-side **EN / 中文** language toggle (defaults to
English; remembers your choice). No build step, no tracking, no dependencies.
